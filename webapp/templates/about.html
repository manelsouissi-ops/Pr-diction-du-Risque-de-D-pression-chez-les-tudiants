À propos du projet

Contexte

La dépression est l'un des problèmes de santé mentale les plus répandus chez les étudiants. Ce projet utilise l'apprentissage automatique pour analyser les facteurs de risque et fournir une évaluation préventive basée sur des données réelles.

Dataset

Le dataset Student Depression Dataset contient 27 901 observations collectées auprès d'étudiants indiens avec 18 variables couvrant leurs conditions académiques, professionnelles et personnelles.

  • Source : Kaggle — Student Depression Dataset
  • Taille : 27 901 lignes × 18 colonnes
  • Cible : Dépression (binaire : 0 / 1)
  • Déséquilibre : 58.5% déprimés / 41.5% non déprimés

Méthodologie

  1. Nettoyage et exploration des données (EDA)
  2. Encodage des variables catégorielles (one-hot, ordinal, binaire)
  3. Standardisation des variables numériques (StandardScaler)
  4. Entraînement LightGBM avec gestion du déséquilibre (is_unbalance=True)
  5. Évaluation : Accuracy, F1, AUC-ROC, matrice de confusion
  6. Interprétation : importance des features + SHAP values

Avertissement

⚠️ Ce modèle est un outil d'aide à la sensibilisation, pas un outil de diagnostic médical. En cas de détresse psychologique, consultez un professionnel de santé qualifié.
